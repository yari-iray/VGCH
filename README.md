# VGCH

## Overview:
The VGCH system was built using TypeScript. The Front-end uses VueJS and rdflib to parse and display the ontology. 

The back-end uses ExpressJS. Tokens are not currently not persisted when deployed to Vercel because of its read-only enforcement.

## Contributions:
Community contributions are highly encouraged, both in the site code and the site contents themselves. Please open an issue or pull request.
